机器人的基础法则

为了防止机器人伤害人类，科幻作家阿西莫夫于1940年提出了“机器人三原则”

  • 第一，机器人不能伤害人类
  • 第二，机器人应遵守人类的命令，与第一条违背的命令除外
  • 第三，机器人应该能保护自己，与第一条违背的命令除外

这是机器人的伦理和底线，比人类的高了太多了。
